# 🏥 Vena — Synthetic Patient Data & Clinical Evidence Agent

> A Google ADK-powered AI agent that combines a **Synthea synthetic patient dataset** in BigQuery with **peer-reviewed medical literature** from PubMed — purpose-built for clinical dashboard and reporting workflows.

---

## What is Vena?

Vena is an intelligent clinical agent built using [Google's Agent Development Kit (ADK)](https://google.github.io/adk-docs/) that gives clinicians and healthcare analysts a conversational interface into two powerful data sources:

| Toolset | Source | What It Does |
|---|---|---|
| **BigQuery MCP** | Synthea synthetic patient dataset | Queries population-level patient data — demographics, diagnoses, medications, encounters, vitals, costs, and more |
| **PubMed MCP** | NCBI PubMed via `pubmed.caseyjhand.com` | Retrieves and synthesises peer-reviewed clinical evidence to support or contextualise data findings |

The agent is designed for **non-technical clinical users** — you ask questions in plain language, and Vena handles the SQL, the literature search, and the synthesis.

> ⚠️ **Important:** All patient data is entirely **synthetic**, generated by [Synthea™](https://synthetichealth.github.io/synthea/). No real patient information is used or stored.

## Dataset Overview

The Synthea dataset contains **16 tables** covering a complete simulated patient population:

| Table | Description |
|---|---|
| `patients` | Demographics, identifiers, lifetime healthcare costs |
| `encounters` | Every clinical visit with class, cost, and payer |
| `conditions` | Diagnoses with onset/resolution dates (SNOMED-CT) |
| `medications` | Prescriptions, costs, and payer coverage (RxNorm) |
| `observations` | Vital signs and lab results (LOINC) |
| `procedures` | Surgical and clinical procedures (SNOMED-CT) |
| `immunizations` | Vaccines administered (CVX) |
| `allergies` | Diagnosed allergies per patient |
| `careplans` | Active and historical care plans |
| `devices` | Implanted/attached devices with FDA UDI |
| `imaging_studies` | DICOM imaging metadata |
| `organizations` | Hospitals and clinics |
| `providers` | Clinicians with speciality and affiliation |
| `payers` | Insurance payers with financial aggregates |
| `payer_transitions` | Insurance coverage history per patient |
| `supplies` | Medical supplies consumed per encounter |

## Setup & Installation

### 1. Clone the repository

```bash
git clone https://github.com/H2S-HackChamps/vena.git
cd vena
```

### 2. Create a virtual environment and activate it

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure environment variables

Run the environment setup script. This script enables the BigQuery and Google Maps APIs, and creates a .env file with your Project ID and Maps API Key.

```bash
chmod +x setup/setup_env.sh
./setup/setup_env.sh
```

> You can obtain a free NCBI API key at [https://www.ncbi.nlm.nih.gov/account/](https://www.ncbi.nlm.nih.gov/account/)

### 5. Load the Synthea dataset into BigQuery

Run the BigQuery setup script. This script automates creating the Cloud Storage bucket, uploading data, and provisioning the BigQuery dataset and tables.:

```bash
chmod +x ./setup/setup_bigquery.sh
./setup/setup_bigquery.sh
```

### 6. Authenticate with Google Cloud

```bash
gcloud auth application-default login
```

### 7. Run the agent

```bash
cd agents/

adk run .
# or for the web UI:
adk web --allow_origins 'regex:https://.*\.cloudshell\.dev'
```

## Sample Queries

Below are sample questions organised by category that you can ask Vena directly in plain language.

---

### 👥 Patient Demographics

> *"How many patients are in the dataset, and what is the breakdown by gender and race?"*

> *"What is the age distribution of currently living patients, grouped into 10-year brackets?"*

> *"Which cities have the highest number of patients, and what are their average lifetime healthcare expenses?"*

---

### 🏥 Encounters & Utilisation

> *"What are the most common types of clinical encounters, and how many unique patients does each encounter class serve?"*

> *"Which organisation has the highest number of encounters, and who are their top 5 providers by utilisation?"*

> *"Show me the monthly trend of emergency encounters over the entire dataset period."*

---

### 🩺 Conditions & Diagnoses

> *"What are the top 10 most prevalent conditions in the dataset by number of affected patients?"*

> *"Which conditions have the longest average duration from onset to resolution?"*

> *"How many patients have more than 3 simultaneous active conditions, and what condition combinations appear most frequently together?"*

---

### 💊 Medications

> *"What are the top 10 most prescribed medications by number of unique patients?"*

> *"Which medications have the highest total cost across the dataset, and what percentage is covered by payers?"*

> *"Show me patients who have been on the same medication for more than 2 years, grouped by medication name."*

---

### 🔬 Observations & Vitals

> *"What is the average BMI broken down by gender and age group?"*

> *"Show me the distribution of systolic blood pressure readings across the patient population."*

> *"Which patients have a recorded HbA1c above 6.5%, and what conditions do they have on record?"*

---

### 💉 Immunizations & Preventive Care

> *"What percentage of patients have received a flu vaccine at least once?"*

> *"Show me the immunization coverage rate by race and ethnicity."*

---

### 💰 Costs & Insurance

> *"Which payer covers the highest percentage of total claim costs across all encounters?"*

> *"What is the average out-of-pocket cost per encounter type for uninsured patients?"*

> *"Show me patients who have switched insurers more than twice, and how their healthcare costs changed across each coverage period."*

---

### 📚 Evidence-Based Clinical Questions (PubMed)

> *"What does current evidence say about first-line treatment for Type 2 Diabetes?"*

> *"Are there any recent systematic reviews on the effectiveness of statins for cardiovascular risk reduction?"*

> *"What does the literature say about the long-term outcomes of patients with chronic sinusitis?"*

---

### 🔗 Combined Data + Evidence Questions

These are the most powerful query type — Vena will pull from both toolsets and present them together:

> *"Our dataset shows a high prevalence of hypertension — what does the evidence say about population-level interventions for blood pressure management?"*

> *"Many patients in this dataset are on Etonogestrel implants for over a year. What does the literature say about long-term safety and efficacy?"*

> *"We're seeing a spike in acute bronchitis diagnoses in winter months. Is there peer-reviewed evidence linking seasonal patterns to acute bronchitis incidence?"*

> *"Which conditions in our patient population have the weakest evidence base for their standard treatments?"*

## Guardrails & Responsible Use

- **Synthetic data only.** All patient records are generated by Synthea™ and contain no real patient information.
- **Not for clinical decision-making on real patients.** Vena is a research and workflow tool — PubMed summaries are informational and should be validated by qualified clinicians before acting on them.
- **PII fields are suppressed.** The agent will not display raw SSN, Drivers License, or Passport fields from the dataset.
- **Evidence is cited.** Every clinical claim sourced from PubMed includes the study type and PMID so findings can be independently verified.
